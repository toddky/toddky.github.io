Settings

  • Sound & vibration -> Vibration & haptics -> Touch feedback
  • Display
    • Dark theme -> Enabled
    • Display size and text -> Display size -> 1
  • Wallpaper & style
    • Shortcuts -> Left shortcut -> Flashlight
    • Show notifications on the lock screen -> Disabled
  • Battery -> Battery percentage -> Enable
  • About phone -> Build number -> Developer options
  • Security & privacy
    • (Main user only) Security -> Security & privacy -> Private Space
    • More security & privacy -> App pinning -> On
  • System
    • Gestures -> Navigation mode
    • Users -> Send notifications to current user -> Enable
    • Developer options -> Window animation scale -> 0.5
    • Developer options -> Transition animation scale -> 0.5
  • Accessibility
    • Timing controls -> Touch & hold delay -> Short
  • ‼️ Fix unable to place/receive phone calls
    • Settings -> System -> Reset options > Reset Mobile Network Settings
