League of Legends

Settings

  • Borderless window
  • 25 Settings for Season 14
    • Game
      • Enable “Clamp cast target location within max range”
      • Disable “Move camera on revive”
    • Quick cast
      • “Quick Cast All” (Hotkeys)
      • Rebind normal cast to Shift+<ability> so you can still see the range indicator (Hotkeys)
    • Attack move
      • Enable “Attack move on cursor” (Game)
      • Enable “Show Attack Range” (Interface)
    • Target Champions Only (Hotkeys) Treat “Target Champions Only” as a toggle (Game)
    • F key cycling
      • Rebind “Select Ally 1-4” to Z, X, C, V (Hotkeys)
    • Chat
      • Enable “Show timestamps”
      • Set “Change Chat Visibility” to “Same Team”
      • In every game type /mute all and /mutepings all
    • Interface
      • Set “HUD Scale” to 0
      • Set “Map Scale” to 50-65
      • Enable “Show Loss of Control UI”
      • Set “Show Names Above Healthbar” to “Champion Name”
      • Enable “Champion highlight on camera center”
      • Enable “Enable Line Missile display”
      • Enable “Show Spell costs”
      • Disable “Show Off-Screen Pings”
      • Set “Emote Buuble Display” to Off
      • Set “Emote Size” to Small
      • Enable “Mute enemy Emotes”
      • Set “Eternals Milestone Display” to “None”
    • Video
      • Disable “Wait for Vertical Sync” (Video)
      • “Effects Quality” should be at least Medium
      • “Character Quality” should be at least Medium
      • “Environment Quality” should be Low or High because there is no difference between Low vs Medium or High vs Very High
      • “Shadow Quality” can be turned Off
      • Enable “Hide eye candy”
      • Enable “Color blind mode”
    • Sound
      • Disable “Music Volume”
      • Disable “Voice Volume”
      • Disable “Ambience Volume”
      • Lower “Ping Volume” and “Announcer Volume” lower than your “Sound FX Volume”
